## Video Library Web App

Preview Link : https://neo-tube.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/Digambar1411/neo-tube.git
```

## Features Available.

- Home Page
- Single Video page
- Likes page
- playlist page
- Watch Later page
- History page
- Authentication
  - *Login*
  - *Signup*
  - *Guest Login*
- Resposive Web app

## Tech Stack

**Client:** React, Context API, React Router v6.

**Server:** Mockbee.
